Command List

Help Commands

Usage Description
_help commands To get help for other helpfull commands
_help info To get help for information commands
_help memes To get help for meme commands
_help games To get help for game commands
_help search To get help for search commands
_help pokemon To get help for pokemon commands
_help admin To get help for admin commands
_help fun To get help for fun commands
_help aki To get help for character guesser(akinator) commands

information commands

Usage Description
_serverinfo Get information about the server
_count members To get help for information commands
_avatar To get or see your avatar
_avatar (member id or mention) Get your information.
_userinfo (mention or member id) Get information about mentioned user.

Meme commands

Usage Description
_joke To get a text based meme .
_meme To get image meme especially dank memes.

Game commands

Usage Description
_tictac (mention) Challenge your opponent for a game of tic tac toe .
_profile To get your profile data in tic-tac-toe.
_lb To get the global leaderboard for tic-tac-toe .

Music commands

Usage Description
_play (song name) Plays the song which you passed .
_pause Pause the current song.
_resume Resume the song which was paused .
_stop Stops the current song .
